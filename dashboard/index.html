Analytics — GA4 Setup Guide

How to connect Google Analytics 4

STEP 1 — CREATE GA4 PROPERTY
Go to analytics.google.com → Admin → Create Property → choose "Web" → enter your site URL.
STEP 2 — GET YOUR MEASUREMENT ID
In GA4 → Admin → Data Streams → your stream → copy the ID. It looks like G-XXXXXXXXXX.
STEP 3 — PASTE THIS IN THE <HEAD> OF EVERY HTML PAGE
<script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXX"></script> <script> window.dataLayer = window.dataLayer || []; function gtag(){dataLayer.push(arguments);} gtag('js', new Date()); gtag('config', 'G-XXXXXXXX'); </script>
STEP 4 — VERIFY
Visit your live site, then check GA4 → Reports → Realtime. Data takes 24-48h to fully populate. Also set up Google Search Console for keyword data.

🗃️ Database Tables — SQL

Run in Supabase SQL Editor to enable all CMS features:
-- Posts table create table if not exists posts ( id uuid default gen_random_uuid() primary key, title text, content text, excerpt text, image_url text, category text, tags text[], meta_title text, meta_desc text, slug text, read_time text, status text default 'draft', created_at timestamptz default now(), updated_at timestamptz default now() ); -- Enquiries create table if not exists enquiries ( id uuid default gen_random_uuid() primary key, name text, email text, phone text, service text, city text, message text, status text default 'new', created_at timestamptz default now() ); -- Members create table if not exists members ( id uuid default gen_random_uuid() primary key, name text, email text unique, phone text, tier text default 'free', city text, status text default 'active', joined_at timestamptz default now() ); -- Picks (supports multiple per category) create table if not exists picks ( id uuid default gen_random_uuid() primary key, type text, name text, city text, rating numeric, image_url text, note text, month_label text, status text default 'published', created_at timestamptz default now(), updated_at timestamptz default now() ); -- Site Config create table if not exists site_config ( key text primary key, value text, updated_at timestamptz default now() ); -- RLS alter table enquiries enable row level security; alter table members enable row level security; alter table site_config enable row level security; alter table picks enable row level security; create policy "anon read picks" on picks for select to anon using (true); create policy "auth manage picks" on picks for all to authenticated using (true); create policy "anon insert enq" on enquiries for insert to anon with check (true); create policy "auth manage enq" on enquiries for all to authenticated using (true); create policy "auth manage members" on members for all to authenticated using (true); create policy "anon read config" on site_config for select to anon using (true); create policy "auth manage config" on site_config for all to authenticated using (true);
